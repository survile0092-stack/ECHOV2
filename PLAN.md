# Разрешить бронирование без имени гостя

**Что изменится:**

- При создании или редактировании бронирования поле «Имя гостя» больше не будет обязательным.
- Если имя гостя не указано, оно автоматически сохранится как **«Имя гостя не указано»**.
- Это название будет отображаться везде, где показывается имя гостя: список бронирований, детали бронирования, уведомления, календарь, экспорт данных.
- Кнопка сохранения бронирования станет активной даже без введённого имени (при заполненных дате и домике).
- Возможность ввести имя гостя сохраняется — поле остаётся, просто становится необязательным.
- В случае редактирования имени, то оно, соответственно, меняется
